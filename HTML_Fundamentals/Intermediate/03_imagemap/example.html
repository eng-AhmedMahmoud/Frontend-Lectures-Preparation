HTML Imagemap Example

What is an imagemap?

An imagemap allows you to define clickable areas on an image. Each area can link to different URLs or trigger different actions.

Key Imagemap Elements:

  • <img usemap="#mapname"> - Links the image to a map
  • <map name="mapname"> - Defines the map container
  • <area> - Defines clickable regions with coordinates
  • shape - Can be "rect", "circle", or "poly"
  • coords - Defines the area boundaries
  • href - Specifies the link destination
