Overflow Control

Overflow: Visible

This content is longer than the container and will overflow outside the boundaries, which is the default behavior.

Overflow: Hidden

This content is longer than the container but will be clipped and hidden when it exceeds the boundaries.

Overflow: Scroll

This content is longer than the container and will show scrollbars to allow users to scroll and see all content.
