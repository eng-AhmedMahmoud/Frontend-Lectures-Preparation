Mixins

Reusable code blocks that accept parameters for flexible styling patterns. 

// Responsive breakpoint mixin
@mixin respond-to($breakpoint) {
  @media (min-width: #{$breakpoint}) {
    @content;
  }
}
✓ DRY code principles
✓ Parametric styling

Functions

Custom functions for calculations, color manipulation, and value transformations. 

// Fluid typography function
@function fluid-size($min, $max) {
  @return clamp(#{$min}, 4vw, #{$max});
}

✓ Color manipulations

Contextual Buttons

// Button variant mixin
@mixin button-variant($color, $bg, $border) {
  color: $color;
  background-color: $bg;
  border-color: $border;
  
  &:hover {
    background-color: darken($bg, 7.5%);
    border-color: darken($border, 10%);
  }
}

// Generate button variants
@each $variant, $colors in $button-variants {
  .btn--#{$variant} {
    @include button-variant(
      map-get($colors, 'color'),
      map-get($colors, 'bg'),
      map-get($colors, 'border')
    );
  }
}
